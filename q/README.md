# Chat Web 聊天网站

本项目是一个基于 Vue + TailwindCSS 的聊天网站，后端采用 Express + Socket.IO。

## 功能特性

- 用户注册与登录
- 好友私聊
- 公共聊天室
- 用户昵称为注册时填写
- UI 风格美观，类似 QQ

## 技术栈

- 前端：Vue 3, Vite, TailwindCSS
- 后端：Express, Socket.IO

## 启动方式

### 前端

```bash
npm run dev
```

### 后端

后端将在 `server` 目录下实现，后续会补充启动说明。

---

如需开发或部署，建议使用最新版 Node.js 和 npm。
